## 数据类型

- num 数字类型的父类，既可以是整型也可以是浮点类型
  num a = 1, b = 1.12;
- int 整型
  int a = 1;
- double 双精度类型
  double a = 1.12;
- String 字符串类型
  String a = 'test';
- Boolean 布尔类型
  Boolean a = true, b = false;
- List 类型，称为集合
  List arr = [1, 2, 3];
- Map类型
  Map map = {'key': 'value'};
